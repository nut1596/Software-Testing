## รายละเอียดงาน

2. Alternating Characters  
3. Caesar Cipher  
4. Two Characters  
5. Grid Challenge  

โดยแบ่งโครงสร้างออกเป็น 2 ส่วน คือ

- `code_under_test` : โค้ดหลักที่ถูกทดสอบ  
- `test_code` : โค้ดสำหรับทำ Unit Test
